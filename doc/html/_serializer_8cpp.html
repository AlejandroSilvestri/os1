#define CARGANDO (   ARCHIVO)    ( typeid(ARCHIVO) == typeid(ArchivoEntrada) )

La bifurcación consiste en distinguir si serializa carga o guarda.

Se activa definiendo BIFURCACION, pero requiere c++11 para typeinfo En un método serialize(Archivo...) la macro GUARDANDO(Archivo) devuelve true si se está guardando, false si se está cargando.

Comentar la siguiente línea para desactivar la capacidad de bifurcación

#define INST_EXP (   CLASE)
