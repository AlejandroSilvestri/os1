Detailed Description

Clase de instancia única que se ocupa de mostrar la imagen de la cámara, con marcas verdes sobre los puntos reconocidos.

Utiliza imshow de opencv. Al pie de la imagen describe el estado del sistema. FrameDrawer se accede desde dos hilos paralelos asincrónicos:

  • Tracking::Track invoca el método FrameDrawer::Update avisando que hay un nuevo Frame para mostrar.
  • Viewer::Run invoca el método FrameDrawer::DrawFrame cuando es momento de actualizar la pantalla.

cv::Mat ORB_SLAM2::FrameDrawer::DrawFrame ( float  radio = 1)

Invocado sólo por Viewer::Run para que envíe la imagen del cuadro actual a la pantalla.

Genera la imagen para visualización.

Sobre el cuadro actual procesado dibuja marcas. En el estado NOT_INITIALIZED dibuja las líneas de flujo óptico consideradas para la triangulación inicial. En el estado OK pone marcas verdes sobre puntos del mapa local visualizados, y marcas azules sobre puntos encontrados para VO

Invocado exclusivamente desde Viewer::Run

Invocado sólo por Tracking::Track avisando que hay un nuevo Frame para mostrar.

Track invoca este método avisando que se terminó de procesar un cuadro.
